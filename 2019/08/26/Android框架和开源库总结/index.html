Android框架和开源库总结

本文最后更新于：2021年8月3日 下午

记录一下用过的Android框架和开源库(持续更新中)

通信框架

Http

Volley

Google出品的Android Http通讯框架

Socket

OkSocket

基于阻塞式传统Socket的一款Socket客户端整体解决方案，可以用来管理TCP长连接
特点：

  • 对通讯协议几乎无限制,可以使用PB,可以使用JSON,可以使用XML.只要可以序列化成Byte数组的对象都可以传输。
  • 兼容所有语言写的Socket服务端,解决了Tcp通讯中头疼的粘包拆包问题,断线重连问题,心跳保持问题,分片发送,重定向连接等问题。
  • 支持单工和全双工通讯。

Serial Port

AndroidSerialPort

Android SDK不提供串口通信的API，Google提供了C语言的串口通信文件，但是需要自己来编译。该作者提供了编译好了的该文件，并进一步封装。

简化代码的框架

ButterKnife

通过注解的方式绑定View资源、OnClickListener等

UI框架

图片类

Glide

一个快速高效的Android图片加载库, 可以加载本地图片也可以通过Url获取图片。

特效、动画类

BlurKit

快速生成高斯模糊的图片，提供动态更新模糊效果的View

TextView

HTextView

支持自定义字体的TextView的动画效果

FadingTextView

以一定时间间隔，以淡入淡出的形式更换内容的TextView

Button
