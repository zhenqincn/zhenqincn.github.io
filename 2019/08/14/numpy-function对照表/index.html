记录一下用过的numpy的function

np

np.array

def array(object, dtype=None, copy=True, order='K', subok=False, ndmin=0)

可以根据list创建一个numpy数组
Parameters:

  • object，创建数组依据的对象
  • dtype，数据类型
  • copy，如果为true（默认值），则复制对象。 否则，只有当array返回副本，obj是嵌套序列，或者需要副本来满足任何其他要求（dtype，order等）时，才会生成副本。
  • order，'K'表示行优先，'F'表示列优先
  • subok, 如果为True，则子类将被传递，否则返回的数组将被强制为基类数组（默认）。
  • ndmin, 指定结果数组应具有的最小维数。为满足此要求，将根据需要预先设置形状。

Returns ndarray

def arange(start, end, step)

返回一个返回为$[start, end)$，步长为step的数组列表
Parameters

  • start, number, 开始值，默认为0
  • end, number, 结束值
  • step, 步长

Returns ndarray

np.argmin

def argmin(a, axis=None, out=None)

返回沿某一个轴计算出的最小值的索引。
Parameters

  • a, 输入的数组
  • axis, 默认情况下会将数组展平，否则沿指定轴
  • out, 如果提供了值，结果将会输入该数组，该数组必须与np.argmin的返回结果有相同的shape和dtype

Returns ndarray, shape为a.shape获得的元组移除掉axis对应的轴，例如a.shape=(500, 100 , 20)，则np.argmin(a, axis=1)返回的数组shape为(500, 20)

np.argwhere

def argwhere(a)

返回所有非零元素的索引, 可结合n == x或者n <= x这种bool表达式来筛选一个数组中满足需求的元素的下标
Parameters

  • a, 输入数组

Returns, ndarray, shape=(?, a的维度)，其中每一项表示一个不为零的元素所在的位置$(x_1, x_2, x_3, … , x_n)$。

Examples

a = np.random.randint(0, 10, (2, 3, 4))
print('1:', np.argwhere(a))
print('2:', np.argwhere(a).shape)
print('3:', a == 2)
print('4:', np.argwhere(a == 2))

np.unique

def unique(ar, return_index=False, return_inverse=False, return_counts=False, axis=None)

找到数组的唯一元素。
返回数组的已排序唯一元素。除了唯一元素外，还有三个可选输出：提供唯一值的输入数组的索引，重构输入数组的唯一数组的索引，以及输入数组中每个唯一值出现的次数。
Parameters

Returns: ndarry, 排序后的所有元素都唯一的数组。

np.linalg.norm

def norm(x, ord=None, axis=None, keepdims=False)

计算矩阵或向量的范数
Parameters

  • x，输入，可以是矩阵，也可以是1维
  • ord，范数类型
    向量的范数：
    • ord=2，计算公式为$\sqrt {x_1^2 + x_2^2 + \ldots + x_n^2} $，默认的计算方式同2
    • ord=1，计算公式为$\left| \right| + \left| \right| + \ldots + \left| \right|$
    • ord=np.inf，表示无穷范数,$l$
      矩阵的范数：
    • ord=2，计算公式为$\left| {\lambda E - {A^T}A} \right|$，求特征值，然后求最大特征值得算术平方根
    • ord=1，列和的最大值
    • ord=np.inf，行和的最大值
  • axis
    • axis=1表示按行向量处理，求多个行向量的范数
    • axis=0表示按列向量处理，求多个列向量的范数
    • axis=None表示矩阵范数。
  • keepdims，是否保持矩阵的二维特性

Returns
float或者ndarray

numpy.ndarray.flatten

def flatten(self, order='C')

将目标数组的copy展平为一维数组

Parameters

  • order, 'C'表示行优先, 'F'表示列优先, 'A'表示如果a在内存中是Fortran连续，则为主列顺序，否则为行主要顺序。'K'表示按元素在内存中出现的顺序展平a。

Returns, ndarray, 输入数组的副本，展平为一维。

numpy.ndarry.reshape

def reshape(self, shape, *shapes, order='C')

返回包含具有新形状的相同数据的数组。

Parameters:

  • shape, tuple, 新数组的形状, 若某一维度不确定，可以用-1来替代，让numpy自动推断，但是只能有一个axis为-1
  • order, 'C'表示行优先, 'F'表示列优先, 'A'表示如果a在内存中是Fortran连续，则为主列顺序，否则为行主要顺序。'K'表示按元素在内存中出现的顺序。

Returns: ndarry, 改变形状后的数组
